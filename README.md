# PokeAPI-CLMTPokemons
O projeto Pokemons-CLMT consiste no desenvolvimento de uma aplicação web que permitirá aos usuários criar uma conta, realizar login e acessar uma interface interativa inspirada em uma Pokédex.

Após autenticado, o usuário poderá visualizar informações detalhadas sobre diversos Pokémons, como nome, tipo, habilidades e outras características relevantes obtidas por meio de uma API. Além disso, cada usuário terá a possibilidade de selecionar e vincular um Pokémon favorito ao seu perfil, personalizando sua experiência dentro da plataforma.

O sistema será baseado em uma API de usuários para gerenciamento de autenticação e dados individuais, garantindo que cada usuário tenha acesso às suas próprias preferências e informações salvas.
